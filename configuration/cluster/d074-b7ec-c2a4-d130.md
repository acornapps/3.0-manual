## 2.3 클러스터

프로바이더로부터 할당받은 쿠버네티스 클러스터 정보를 등록, 편집, 삭제할 수 있습니다.

##### a\)    환경설정 → 클러스터  →  정렬 버튼을 클릭하여 원하시는 정렬기준을 선택합니다. ![](/assets/2.3.0 클러스터.png)

| **클러스터 설정 ** | **설명** |
| :--- | :--- |
| + 클러스터 등록 | 클러스터 추가 페이지 이동  |
| 클러스터 | 클러스터 추가 페이지 이동 |
| 노드 | 클러스터 ID값\(자동부여\) |
| CPU | 클러스터 이름\(사용자지정\) |
| 메모리 | 프로바이더 유형\(CUBE/GKE\) |
| 어플리케이션 | 현재 클러스터의 사용 여부 |
| 서버 | 클러스터에서 등록 중인 프로바이더 |
