## 5.5.1 알람 목록

---

알람은 다음 목록에서 발생 조건이 지속 시간 만큼 계속 될 경우 발생한다.

* #### AlertManager

| 알람 ID | **ALM-001** |
| :--- | :--- |
| 중요도 | warning |
| 알람 이름 | AlertmanagerDown |
| 지속 시간 | 5분 |
| 발생 조건 | Alertmanager 메트릭 수집이 안 될 경우 발생 |

| 알람 ID | **ALM-002** |
| :--- | :--- |
| 중요도 | warning |
| 알람 이름 | AlertmanagerFailedReload |
| 지속 시간 | 10분 |
| 발생 조건 | Alertmanager의 설정 변경시, 설정 다시읽기 작업 실패시 발생 |
