### 2.1.2 사용자 편집

---

칵테일 사용자 정보를 편집 하는 기능 이다.

##### a\) 환경설정 → 사용자 → 사용자 액션 메뉴 → 사용자 편집 → 수정완료 후 수정버튼을 클릭한다.![](/assets/2.3.0 사용자 편집.png)

##### b\) 사용자 아이디를 제외한 이름, 상태 및 권한 수정이 가능하다.![](/assets/2.3.0 사용자 편집2.png)

| 상태 | **설명** |
| :--- | :--- |
| Active | 사용자 활성화 |
| Inactive | 사용자 비활성 |
