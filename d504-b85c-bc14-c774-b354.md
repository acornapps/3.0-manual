# 프로바이더 관리

---

### 1 프로바이더 계정 생성

1. 환경설정
2. 프로바이더
3. 등록\(+ 아이콘\)
4. 이름, 유형, 설명 등의 정보를 입력 후 생성 버튼 클릭

### 2 프로바이더 계정 조회

1. 환경설정
2. 프로바이더

### 3 프로바이더 계정 수정

1. 환경설정
2. 프로바이더
3. 프로바이더 계정 목록중 해당 계정의 액션아이콘
4. 프로바이더 편집
5. 이름, 유형, 설명 등의 정보를 편집 후 수정 버튼 클릭

### 4 프로바이더 계정 삭제

1. 환경설정
2. 프로바이더
3. 프로바이더 목록중 해당 클러스터의 액션아이콘
4. 프로바이더 삭제
